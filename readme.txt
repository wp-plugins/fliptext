== Description ==

Display your text in mirror mode

For support and further information about fliptext plugin see the plugin homepage at [Fliptext homepage](http://www.telesphore.org/plugins/fliptext "Link to Fliptext homepage").

== Installation ==

1. Download the plugin
1. Upload the `fliptext` folder to the `/wp-content/plugins/` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Write an article and click Fliptext button to flip your selected text
